# UI 요소 및 액션 목록 (수동 붙잡힘 시스템)

## 1. 스플래시 화면
- **로고 이미지**
- **앱 이름 텍스트**: "경찰과 도둑"
- **로딩 인디케이터**

---

## 2. 튜토리얼 화면 (첫 실행시)
- **설명 이미지** (슬라이드별)
- **설명 텍스트**
- **진행 인디케이터** (1/4, 2/4, 3/4, 4/4)
- **[다음] 버튼** → 다음 슬라이드로 이동
- **[건너뛰기] 버튼** → 튜토리얼 종료, 닉네임 설정으로 이동

---

## 3. 닉네임 설정 화면
- **안내 텍스트**: "게임에서 사용할 닉네임을 입력하세요"
- **닉네임 입력 필드** (2-8자 제한)
- **[시작하기] 버튼** → 닉네임 저장, 홈 메뉴로 이동

---

## 4. 홈 메뉴 화면
- **햄버거 메뉴 아이콘** → 설정/프로필 메뉴 열기
- **프로필 아이콘** → 내 기록 화면으로 이동
- **환영 텍스트**: "환영합니다, {닉네임}님"
- **[새 게임 만들기] 버튼** → 게임 설정 화면으로 이동
- **[게임 참가하기] 버튼** → 방 코드 입력 화면으로 이동
- **[내 기록] 버튼** → 내 기록 화면으로 이동
- **[설정] 버튼** → 설정 화면으로 이동

---

## 5. 게임 설정 화면 (새 게임)
- **[← 뒤로가기] 버튼** → 홈 메뉴로 복귀
- **제목**: "새 게임 만들기"
- **게임 이름 입력 필드**
- **플레이 시간 선택 버튼 그룹**
  - **[15분] 버튼** → 15분 선택
  - **[30분] 버튼** → 30분 선택 (기본값)
  - **[60분] 버튼** → 60분 선택
- **위치 공개 주기 선택 버튼 그룹**
  - **[1분] 버튼** → 1분 선택
  - **[3분] 버튼** → 3분 선택 (기본값)
  - **[5분] 버튼** → 5분 선택
- **포획 판정 거리 선택 버튼 그룹**
  - **[3m] 버튼** → 3m 선택
  - **[5m] 버튼** → 5m 선택 (기본값)
  - **[10m] 버튼** → 10m 선택
- **[다음 단계] 버튼** → 영역 설정 화면으로 이동

---

## 6. 영역 설정 화면
- **[← 뒤로가기] 버튼** → 게임 설정 화면으로 복귀
- **제목**: "놀이 영역 설정"
- **지도 뷰** (현재 위치 중심)
- **내 위치 마커**
- **영역 원형 오버레이** (반투명)
- **반경 슬라이더** (100m ~ 1000m)
  - 드래그 시 → 영역 반경 실시간 조정
- **반경 표시 텍스트**: "반경: {value}m"
- **안내 텍스트**: "놀이 영역 중심을 터치하세요"
- **[영역 확정] 버튼** → 방 생성, 방 코드 생성 화면으로 이동

---

## 7. 방 코드 생성 완료 화면
- **제목**: "방이 생성되었습니다!"
- **방 코드 표시 영역** (큰 글씨)
- **[복사] 버튼** → 방 코드를 클립보드에 복사
- **[QR 코드 보기] 버튼** → QR 코드 팝업 표시
- **[공유] 버튼** → 시스템 공유 기능 (카톡, 문자 등)
- **안내 텍스트**: "친구들과 코드를 공유하세요"
- **[대기실로 이동] 버튼** → 대기실 화면으로 이동

---

## 8. 방 코드 입력 화면
- **[← 뒤로가기] 버튼** → 홈 메뉴로 복귀
- **제목**: "게임 참가하기"
- **방 코드 입력 필드** (6자리)
- **[참가하기] 버튼** → 방 코드 검증 후 대기실 입장
- **[QR 스캔] 버튼** → QR 스캐너 실행, 스캔 성공 시 자동 입장

---

## 9. 대기실 화면 - 방장

### 헤더
- **[← 나가기] 버튼** → 방 나가기 확인 팝업 → 홈 메뉴로 복귀
- **방 이름 표시**
- **참가자 수 표시**: "{현재}/{최대}"

### 참가자 목록
- **참가자 카드** (각 참가자별)
  - 프로필 아이콘
  - 닉네임
  - 역할 선택 버튼 **[👮 경찰] / [🏃 도둑]** → 클릭 시 역할 전환
  - 준비 상태 표시 (✓준비)
  - **[강퇴] 버튼** (방장만, 다른 참가자에게만 표시) → 해당 참가자 강퇴

### 하단
- **채팅 미리보기 영역**
- **[💬 채팅] 버튼** → 채팅 전체화면 열기
- **[⚙️ 설정 변경] 버튼** → 게임 설정 화면으로 복귀 (수정 가능)
- **[🎮 게임 시작] 버튼** (방장만) → 모든 참가자 준비 확인 후 게임 시작

---

## 10. 대기실 화면 - 참가자

### 헤더
- **[← 나가기] 버튼** → 방 나가기, 홈 메뉴로 복귀
- **방 이름 표시**
- **참가자 수 표시**

### 참가자 목록
- **참가자 카드** (각 참가자별)
  - 프로필 아이콘
  - 닉네임
  - 역할 표시 (👮 경찰 / 🏃 도둑)
  - 준비 상태 표시
  - 방장 표시 (왕관 아이콘)

### 하단
- **채팅 미리보기 영역**
- **[💬 채팅] 버튼** → 채팅 전체화면 열기
- **[✓ 준비완료] / [준비취소] 토글 버튼** → 준비 상태 전환

---

## 11. 도둑 플레이 화면 ⭐

### 헤더
- **타이머 표시**: "⏱️ {남은시간}"
- **역할 표시**: "🏃 도둑"
- **[⚙️ 메뉴] 버튼** → 게임 중 메뉴 (설정, 포기 등)

### 지도 영역
- **지도 뷰** (전체 화면의 60%)
- **내 위치 마커** (파란 점)
- **영역 경계선** (원형)
- **동료 도둑 위치 마커** (작은 파란 점)
- **경찰 위치 마커** (위치 공개 시에만, 빨간 점, 5초간 표시)

### 정보 패널
- **다음 위치 공개 카운트다운**: "다음 위치 공개: {시간} ⏳"
- **현재 상태 표시**
  - "✅ 안전 영역 내"
  - "⚠️ 영역 경계 근처"
  - "🚨 경찰 근처!" (경찰이 30m 이내일 때)
- **생존자 수 표시**: "생존자: {수}명"

### 하단 버튼
- **[❌ 잡혔어요] 버튼** (중요!) → 경찰에게 실제로 터치당했을 때 누름 → 포획 확인 팝업 표시
- **[💬 팀 채팅] 버튼** → 도둑 팀 채팅방 열기
- **[📍 현재 위치 새로고침] 버튼** → GPS 위치 수동 업데이트

---

## 12. 도둑 - 위치 공개 순간 (알림)
- **전체 화면 빨간 오버레이** (1초간)
- **알림 텍스트**: "🚨 위치가 공개되었습니다!"
- **다음 공개 카운트다운 리셋**: "다음 공개까지: 3:00"
- **진동 알림**
- **알림음**

---

## 13. 도둑 - "잡혔어요" 확인 팝업
- **팝업 제목**: "정말 잡혔나요?"
- **확인 텍스트**: "경찰에게 터치당하셨나요?"
- **잡은 경찰 선택 드롭다운** → 어느 경찰에게 잡혔는지 선택
- **[예, 잡혔어요] 버튼** → 포획 확정, 포획됨 화면으로 전환
- **[아니요, 취소] 버튼** → 팝업 닫기

---

## 14. 경찰 플레이 화면

### 헤더
- **타이머 표시**: "⏱️ {남은시간}"
- **역할 표시**: "👮 경찰"
- **[⚙️ 메뉴] 버튼** → 게임 중 메뉴

### 지도 영역
- **지도 뷰**
- **내 위치 마커** (파란 점)
- **동료 경찰 위치 마커** (초록 점)
- **도둑 위치 마커** (위치 공개 시에만, 빨간 X, 5초간)

### 도둑 목록 패널
- **도둑 카드** (각 도둑별)
  - 닉네임
  - 현재 거리 표시: "{거리}m"
  - 방향 화살표 (나침반 기준)
  - **[안내] 버튼** → 해당 도둑 위치로 지도 중심 이동 + 경로 안내

### 정보 표시
- **다음 위치 공개 카운트다운**
- **남은 도둑 수**: "남은 도둑: {수}명"
- **내가 잡은 수**: "내 포획: {수}명"

### 하단 버튼
- **[💬 팀 채팅] 버튼** → 경찰 팀 채팅방 열기
- **[📍 가장 가까운 도둑] 버튼** → 가장 가까운 도둑 위치로 지도 이동

## 17. 관전 모드 화면

### 헤더
- **타이머 표시**
- **제목**: "👁️ 관전 중"
- **[← 나가기] 버튼** → 게임 나가기 확인 팝업

### 지도 영역
- **지도 뷰**
- **모든 경찰 위치 마커** (초록 점)
- **모든 생존 도둑 위치 마커** (파란 점)
- **영역 경계선**

### 현재 상황 패널
- **생존 도둑 수 표시**
- **생존 도둑 목록**
  - 닉네임
  - 상태 (도주중/숨는중 등)
- **잡힌 도둑 목록** (회색 처리)

### 하단 버튼
- **[💬 채팅 보기] 버튼** → 전체 채팅 (읽기 전용)
- **[카메라 전환] 버튼** → 특정 플레이어 시점으로 지도 이동

## 18. 게임 결과 화면

### 상단
- **제목**: "🎉 게임 종료"
- **승리 팀 표시**: "👮 경찰 승리!" / "🏃 도둑 승리!"
- **게임 요약**
  - "모든 도둑 포획 - 소요시간: {시간}" (경찰 승리시)
  - "생존자 {수}명 - 시간 종료" (도둑 승리시)

### MVP 영역
- **제목**: "🏆 MVP"
- **MVP 카드**
  - 프로필 아이콘
  - 닉네임
  - 역할
  - 주요 기록

### 상세 기록 목록
- **탭 버튼 그룹**
  - **[전체] 탭** → 모든 참가자
  - **[경찰] 탭** → 경찰만
  - **[도둑] 탭** → 도둑만

- **참가자 기록 카드** (각 참가자별)
  - 순위
  - 닉네임
  - 역할
  - 주요 기록 (포획 수 / 생존 시간)

### 하단 버튼
- **[🗺️ 경로 보기] 버튼** → 이동 경로 재생 화면으로 이동
- **[📸 결과 공유] 버튼** → 결과 이미지 생성 후 공유
- **[💬 채팅] 버튼** → 게임 채팅 기록 보기
- **[🔄 다시 하기] 버튼** → 같은 설정으로 새 게임 생성
- **[🏠 홈으로] 버튼** → 홈 메뉴로 이동

---

## 19. 이동 경로 재생 화면

### 헤더
- **[← 뒤로가기] 버튼** → 결과 화면으로 복귀
- **제목**: "이동 경로"

### 지도 영역
- **지도 뷰**
- **각 플레이어 이동 경로** (컬러 선)
- **재생 위치 마커** (현재 재생 시점)

### 재생 컨트롤
- **[▶️ 재생 / ⏸️ 일시정지] 버튼** → 경로 재생 토글
- **재생 진행 슬라이더** → 드래그로 특정 시점 이동
- **시간 표시**: "{현재시간} / {총시간}"
- **재생 속도 버튼**: [1x] [2x] [5x] → 재생 속도 변경

### 표시 옵션
- **플레이어 선택 체크박스** (각 플레이어별)
  - 체크 → 해당 플레이어 경로 표시
  - 체크 해제 → 경로 숨김

### 하단 버튼
- **[📸 스크린샷] 버튼** → 현재 화면 캡처 후 저장/공유

---

## 20. 팀 채팅 화면

### 헤더
- **[← 뒤로가기] 버튼** → 이전 화면으로 복귀
- **제목**: "팀 채팅 (도둑)" / "팀 채팅 (경찰)"
- **참가자 수 표시**

### 메시지 영역
- **메시지 리스트** (스크롤 가능)

    - 발신자 닉네임 (내 메시지는 오른쪽 정렬)
    - 메시지 내용
    - 전송 시간

### 입력 영역
- **메시지 입력 필드**
- **[이모지] 버튼** → 이모지 선택 패널 열기
- **[전송] 버튼** → 메시지 전송, 입력창 초기화

## 21. 설정 화면

### 헤더
- **[← 뒤로가기] 버튼** → 홈 메뉴로 복귀
- **제목**: "설정"

### 알림 섹션
- **섹션 제목**: "알림"
- **[위치 공개 알림] 토글 스위치** → ON/OFF 전환
- **[포획 알림] 토글 스위치** → ON/OFF 전환
- **[채팅 알림] 토글 스위치** → ON/OFF 전환

### 지도 섹션
- **섹션 제목**: "지도"
- **지도 스타일 선택**
  - **[일반] 버튼** → 일반 지도
  - **[위성] 버튼** → 위성 지도
- **GPS 정확도 슬라이더** → 배터리 vs 정확도 조절

### 배터리 섹션
- **섹션 제목**: "배터리"
